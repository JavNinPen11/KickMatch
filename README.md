# KickMatch
Repositorio de desarrollo del proyecto de fin de grado de desarrollo de aplicaciones web

# ¿Cómo iniciar la aplicacion?
Abre 2 terminales, una debe estar localizada en la carpeta backend y la otra en la carpeta frontentd.
En la terminal del backend utilizarás el comando npm run dev para iniciar la lógica de servidor
En la terminal del frontend utilizarás el comando npm start, el cual te iniciara la pagina de react en localhost:3000
